# `mask01.py` —— 流域掩膜生成器

本脚本读取包含 Pfafstetter 流域编码的 GeoTIFF 文件，根据指定的流域编码提取该流域的最小外接矩形（向外扩展一个像元），生成一个二值掩膜 TIFF 文件。

---

## 1. 功能概述

- 输入：一个流域编码 TIFF 文件（单波段，像元值为整型或浮点型编码）
- 处理：
  - 定位所有等于目标 `pfbas_code` 的像元
  - 计算这些像元的最小外接矩形（行列范围）
  - 向外扩展 **1 个像元**（上下左右各扩 1 格，不超出原图边界）
  - 裁剪出该矩形区域
- 输出：二值掩膜 GeoTIFF
  - **值 1**：原流域所在像元
  - **值 0**：矩形内非流域像元
  - **无数据值**：默认为 `255`（适合 `uint8` 类型）

---

## 2. 使用方法

### 直接运行（示例）

```bash
python mask01.py
```

脚本将：

- 处理当前目录下的 `PFBAS8.tif` 文件
- 提取编码为 `1215` 的流域
- 输出 `output_mask.tif`
- 打印输出文件中的唯一值（验证是否包含 0、1、255）

### 作为模块调用

```python
from mask01 import extract_watershed_bbox_with_mask

extract_watershed_bbox_with_mask(
    tif_path="your_input.tif",
    pfbas_code=12345,
    output_tif_path="your_output.tif",
    nodata=255
)
```

---

## 3. 函数参数说明

| 参数  | 类型  | 说明  |
| --- | --- | --- |
| `tif_path` | `str` | 输入的流域编码 TIFF 文件路径 |
| `pfbas_code` | `int / float` | 目标流域的 Pfafstetter 编码（数值类型） |
| `output_tif_path` | `str` | 输出的二值掩膜 TIFF 文件路径 |
| `nodata` | `int` | 输出文件的无数据值，默认 `255` |

---

## 4. 输出文件特性

- **格式**：GeoTIFF
- **压缩**：LZW（有效减小文件体积）
- **数据类型**：`uint8`
- **地理变换**：保留原始投影和仿射变换，仅更新到裁剪后的区域
- **扩展方式**：向外扩展一个像元，并自动截断至原始栅格边界内

## 6. 代码核心逻辑

1. **打开输入文件，读取第一波段数据**
2. **定位目标编码像元** → 获得行列索引
3. **计算外接矩形边界** → 向外扩展 1 像元 → 限制边界
4. **创建裁剪窗口** → 读取窗口内数据
5. **生成输出数组**（全 0）→ 将窗口内等于目标编码的位置设为 1
6. **更新地理变换和元数据** → 写入输出 TIFF

---

## 7. 注意事项

- **内存**：脚本一次性读取整个栅格，对于超大 TIFF 可能内存不足（可后续改为分块处理）。
- **编码类型**：`pfbas_code` 必须为数值类型（`int` 或 `float`），不支持字符串。若需处理带前导零的编码（如 `02080601`），请在外层转换为整数后传入。
- **NoData 处理**：输入文件的 `nodata` 值不会被特殊处理；若编码等于 `nodata`，可能导致错误识别。建议输入文件中目标编码与 `nodata` 不同。
- **输出验证**：脚本运行后会打印输出数组的唯一值，可用于快速检查（正常应包含 `0`、`1`，可能包含 `255` 作为无数据值）。
